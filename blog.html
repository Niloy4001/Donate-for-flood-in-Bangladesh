Important FAQ

1. What is the Document Object Model (DOM)?

Document Object Model (DOM) is a programming interface that shows HTML page as a tree of object, where each and every element,attributes,text etc are the part of that object tree. Using DOM javascript can interact the HTML page. Because javascript works with it's fundamental concept like Function,Array,Object, as DOM shoes HTML page as tree of object , that's why javascript can interact or dynamically change HTML page's element,attributes,style,text etc.

2. How do you select an element from the DOM?

There are several ways to select an element from DOM.
1. document.getElementById(), method selects a element which has a unique id attribute.For instance document.getElementById('myInfo'); will select that specific element which contains the id attribute with the value of myInfo.
2. document.querySelector(), method selects a element the first element that matches a CSS selector. For example , document.querySelector('.myInfo'); will select the first element of those who contain 'myInfo' CSS class selector.
There are multiple ways to select multiple element from DOM. They are,
1.document.getElementsByClassName(); will selects multiple element who contain same class.
2.document.getElementsByTagName(); selects those specific element who has same tag name
3.document.querySelectorAll(); selects them who has same css selector. For example document.querySelectorAll('.myInfo'); will select those element who has the class attribute with the value of 'myInfo'.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where I can attach a single event listener to a parent element instead of adding multiple event listeners to individual child elements. The event listener on the parent catches events that bubble up from the child elements. For example I have a lot of button on a ul tag , then I can add an eventListener to parent ul element instead of adding multiple eventListener on individual child button element.
Why it is Important:
1.Because of low number of event handler the codes perform better.
2. It makes code easier or simplify the code.

4. How do you manipulate an element's attributes and styles using the DOM?

There are many ways or method to manipulate attribute, they are
1.setAttribute() , using this method I can set any attribute to any element
2.removeAttribute() , using this method I can remove any attribute from any element
3.getAttribute() , using this method I can check any attribute is present in any element or not.

On the other hand there several method to manipulate style, they are like
1.element.style , using this method I can add CSS style in any element
2.classList, by this method I can remove or add any class to any element, for instance we are going to add or remove the class 'text-center' from an element,then we use the method like this, classList.add('text-center') or classList.remove('text-center').
